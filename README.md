# HERB SHOP
technologies: 
```
css
html
JavaScript
```

Cel i zakres projektu:
```
▪ celem projektu jest wykonanie witryny internetowej o dowolnej tematyce (szata graficzna nie
podlega ocenie)
▪ strona internetowa powinna mieć strukturę w standardzie HTML5 i zawierać następujące
elementy:
✓ grafikę oraz treść (nie może być stosowany „wypełniacz”: Lorem ipsum …)
✓ nagłówek, menu, stopkę oraz sekcje; sekcje w ilości uzależnionej od tematyki
✓ menu powinno składać się z trzech opcji: strona główna, formularz oraz galeria zdjęć
✓ formularz powinien:
a. umożliwiać wprowadzenie następujących danych:
• nazwisko
• imię
• adres e-mail
• ulica wraz z numerem domu/mieszkania
• kod pocztowy
• miasto
b. sprawdzać poprawność (pod kątem formatu) wszystkich wprowadzonych danych
(walidacja danych); sprawdzenie może odbywać się w trakcie wprowadzania danych
lub po naciśnięciu odpowiedniego przycisku, po wprowadzeniu wszystkich danych
c. umożliwiać wyświetlenie podpowiedzi (np. w postaci „chmurki”) w momencie
„kliknięcia” w wybrane pole formularza; „chmurka” powinna informować jaki jest
wymagany sposób (tzn. format) wprowadzania danych
✓ galeria zdjęć (min. 12) z możliwością automatycznego powiększania po najechaniu myszą
na miniaturę zdjęcia. 
```
